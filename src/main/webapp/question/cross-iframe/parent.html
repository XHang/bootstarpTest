现在测试ifame的跨域问题
你现在应该可以看见浏览器控制台报错了 Refused to display 'xxx' in a frame because it set 'X-Frame-Options' to 'deny'. 这就是ifram的跨域问题了，想要解决此类问题 需要在页面的响应头上加上X-Frame-Options响应头，并设为 SAMEORIGIN 表示该页面可以在相同域名页面的 frame 中展示。 ALLOW-FROM uri 表示该页面可以在指定来源的 frame 中展示。
